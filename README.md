## Развертывание приложения <br>

# База данных <br>
Прописываем: <br>
```
json-server db.json
```
или <br>
```
npx json-server db.json 
```
# Установка базовых компонентов
Установим на проект node(локально либо глобально как вам удобнее
```
npm i
```
